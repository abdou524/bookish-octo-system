# Ar Transport - Site Web de Convoyage de Véhicules

## À propos du projet

Ce projet est un site web professionnel pour **Ar Transport**, une entreprise spécialisée dans le convoyage de véhicules automobiles à l'échelle nationale et internationale.

## Fonctionnalités

✅ **Navigation intuitive** - Menu de navigation sticky pour accès facile aux sections
✅ **Responsive Design** - Compatible avec tous les appareils (desktop, tablette, mobile)
✅ **Section Accueil** - Banner héroïque avec appel à l'action
✅ **Services** - Présentation de 6 services principaux avec icônes
✅ **Avantages** - 6 raisons de choisir Ar Transport
✅ **Statistiques** - Affichage des résultats et réalisations
✅ **Contact** - Formulaire de contact + informations d'entreprise
✅ **Réseaux Sociaux** - Liens vers les plates-formes sociales
✅ **Footer** - Information de copyright

## Structure du projet

```
ar-transport/
├── index.html      # Page principale (HTML)
├── styles.css      # Feuille de styles (CSS)
├── images/         # Dossier pour les images
└── README.md       # Ce fichier
```

## Services proposés

1. **Transport National** - Livraison sécurisée avec suivi en temps réel
2. **Transport International** - Convoyage vers l'Europe
3. **Logistique Automobile** - Gestion de flotte et stockage
4. **Assurance Complète** - Couverture tous risques
5. **Livraison Rapide** - Délais optimisés
6. **Sécurité Maximale** - GPS et suivi 24/7

## Technologies utilisées

- **HTML5** - Structure sémantique
- **CSS3** - Design moderne avec Flexbox et Grid
- **JavaScript** (optionnel) - Pour animations supplémentaires

## Comment utiliser

1. Clonez ou téléchargez le projet
2. Créez un dossier `images` et ajoutez vos images de véhicules
3. Modifiez les informations de contact (téléphone, email, adresse)
4. Hébergez le site sur un serveur web
